# 크롬 확장 프로그램 설치 방법

잡코리아(JobKorea) 채용공고 페이지에서 잡플래닛(Jobplanet) 기업 리뷰를 바로 확인할 수 있는 크롬 확장 프로그램 설치 가이드입니다.

---

## 1. 설치 파일 준비

프로젝트 루트에 있는 `module-chrome-ext.zip` 파일을 다운로드하거나, 이미 로컬에 있는 경우 그대로 사용합니다.

---

## 2. 압축 해제

`module-chrome-ext.zip` 파일을 원하는 폴더에 압축 해제합니다.

압축 해제 후 아래 파일들이 포함되어 있어야 합니다:

```
module-chrome-ext/
├── manifest.json
├── background.js
├── content.js
├── popup.html
├── popup.js
├── styles.css
└── icons/
```

---

## 3. 크롬 확장 프로그램 로드

### 3-1. 크롬 확장 관리 페이지 열기

크롬 브라우저 주소창에 아래 주소를 입력합니다:

```
chrome://extensions
```

### 3-2. 개발자 모드 활성화

페이지 우측 상단의 **"개발자 모드"** 토글을 켭니다.

### 3-3. 압축 해제된 확장 프로그램 로드

개발자 모드를 켜면 상단에 새 버튼들이 나타납니다.

**"압축해제된 확장 프로그램을 로드합니다."** 버튼을 클릭합니다.

파일 선택 창에서 2단계에서 압축 해제한 `module-chrome-ext` 폴더를 선택합니다.

### 3-4. 설치 확인

확장 프로그램 목록에 **"Crawling Planet"** (또는 유사한 이름)이 표시되면 설치가 완료된 것입니다.

---

## 4. 사용 방법

1. 크롬 브라우저에서 **잡코리아(jobkorea.co.kr)** 채용공고 페이지로 이동합니다.
2. 채용공고 페이지가 로드되면 확장 프로그램이 **자동으로** 해당 회사의 잡플래닛 리뷰를 가져와 페이지에 표시합니다.
3. 별도의 조작 없이 채용공고를 보면서 기업 리뷰를 함께 확인할 수 있습니다.

---

## 5. API 서버 안내

이 확장 프로그램은 **`crawling-planet.cc`** 서버에 연결하여 기업 리뷰 데이터를 제공받습니다.

- 서버가 정상 운영 중일 때만 리뷰 데이터가 표시됩니다.
- 네트워크 연결 상태 및 서버 상태에 따라 데이터 로딩 속도가 달라질 수 있습니다.

---

## 문제 해결

| 증상 | 해결 방법 |
|------|-----------|
| 확장 프로그램이 목록에 나타나지 않음 | 개발자 모드가 켜져 있는지 확인 후, 올바른 폴더(manifest.json이 있는 폴더)를 선택했는지 확인 |
| 잡코리아 페이지에서 리뷰가 표시되지 않음 | 확장 프로그램이 활성화되어 있는지 확인. `chrome://extensions`에서 토글이 켜져 있어야 함 |
| 리뷰 데이터가 없다고 표시됨 | 해당 회사의 리뷰 데이터가 아직 수집되지 않았을 수 있음 |
| 서버 연결 오류 | `crawling-planet.cc` 서버 상태를 확인하거나 잠시 후 다시 시도 |
