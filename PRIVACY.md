# Privacy Policy / 개인정보처리방침

**Jobplanet Review Viewer** Chrome Extension

최종 수정일: 2026-03-29

## 수집하는 정보

본 확장 프로그램은 다음 정보만을 처리합니다:

- **잡코리아 채용공고 페이지의 회사명 텍스트**: 해당 회사의 잡플래닛 리뷰를 조회하기 위해 사용됩니다.

## 수집하지 않는 정보

본 확장 프로그램은 다음 정보를 수집, 저장, 전송하지 않습니다:

- 개인 식별 정보 (이름, 이메일, 전화번호 등)
- 로그인 정보 또는 인증 정보
- 브라우저 방문 기록
- 쿠키 또는 세션 데이터
- 위치 정보
- 키 입력, 마우스 클릭 등 사용자 활동 데이터

## 데이터 처리 방식

1. 사용자가 잡코리아(jobkorea.co.kr) 채용공고 페이지를 방문하면, 페이지에 표시된 회사명을 읽습니다.
2. 읽은 회사명을 crawling-planet.cc API 서버에 전송하여 잡플래닛 리뷰 데이터를 조회합니다.
3. 조회된 리뷰 데이터(평점, 장점, 단점 등)를 채용공고 옆에 표시합니다.
4. 회사명 외의 어떠한 데이터도 서버에 전송되지 않습니다.

## 제3자 제공

수집된 데이터는 제3자에게 판매, 전송, 공유되지 않습니다.

## 데이터 보관

본 확장 프로그램은 사용자 데이터를 영구적으로 저장하지 않습니다. 조회된 리뷰 데이터는 브라우저 탭이 열려 있는 동안만 메모리에 캐시되며, 탭을 닫으면 삭제됩니다.
